# Tinkoff Invest Robot Contest: номинация "Торговый робот для мобильных устройств"

## Реализованная функциональность
- Авторизация с использованием токена
- Автоматическая торговля алгоритмом RSI
- Ведение статистики
- Настройки параметров алгоритма RSI
- Обработка технических ошибок при работе с сетью / Tinkoff API
- Обработка ошибок предметной логики

## Особенности проекта в следующем
- Торговля **одновременно** на **множестве** акций
- Темная / светлая тема
- **Валидация токена** на старте
- Поддержка **расписаний торгов**
- Готовые пресеты "Спокойный" и "Активный" для быстрого запуска без погружения в детали алгоритма
- Гибкая настройка алгоритма RSI "под себя" для опытных пользователей
- Удобный **поиск по акциям** позволяет искать сразу по названию, тикеру или figi, поисковая система предложит максимально точное совпадение
- **Уведомления** в Dock при торговле на нескольких акиях
- Торговля на выбранном счете. Самостоятельно выбирайте счет, на котором будет торговать бот
- Режим эмуляции. В режиме эмуляции система **загрузит истории рыночных котировок** в указанный период времени и проведет тестирование стратегии на исторических данных, а эмуляция ускорит процесс, таким образом пользватель может быстро протестировать настройки алгоритма без лишних задержек.
- Режим песочницы. Отлаживайте работу алгоритма в реальном времени, используя возможности песочницы Tinkoff API
- Режим Тинькофф. **Торгуйте** настоящими средствами **на реальной бирже**
- **Подробная статистика**, включающая в себя: график свечей, график значений rsi, количество купленных и проданных акций за время работы торгового робота, расчет ожидаемой прибыли, а так же консоль событий для самых продвинутых пользователей
- Обработка технических ошибок. При возникновении проблем с сетью / Tinkoff API, бот отправляет несколько retry запросов. А если проблема сохраняется, информирует пользователя в консоле событий
- Обработка ошибок предметной логики. Бот кэширует наличие позиций и средств к покупке, что позволяет избежать лишних запросов в Tinkoff API, при отсутствии средств

## Немного про алгоритм торговли
Приложеиние GoldenBrazierInvest позволяет рабоать с любой торговой стартегией, поддерживающей специальные коллбэки для визуализации.
В качесвте примера был реализован движок [RsiEngine](https://github.com/GoldenBrazier/TinkoffInvestRobot/blob/master/GoldenBrazierInvest/StratagyEngine/RsiEngine.swift), 
работающий на алгоритме RSI. Подробнее про сам алгоритм вы можете прочитать [здесь](https://ru.wikipedia.org/wiki/Индекс_относительной_силы).

- За время тестирования алгоритм продемонстрировал свою надежность, наторговав в плюс как на исторических данных в режиме эмуляции, так и в реальном режиме!

## Требования для сборки проекта
- macOS >= v12
- xCode >= v13

## Зависимости
Все зависимости уже включены в сборку, дополнительно ничего устанавливать не требуется
- Note: xCode может не сразу найти все зависимости при первичной сборке проекта, для решения этой проблемы рекомендуется почистить package cache. File->Packages->Reset Package Cache 
- Note: В проекте используется кастомная версия [TinkoffInvestSDK](https://github.com/nimelehin/TinkoffInvestSwiftSDK)

## Информация для конкурса
